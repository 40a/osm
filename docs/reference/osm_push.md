## osm push

Push item from container

### Synopsis


Push item from container

```
osm push <src> <dest> [flags]
```

### Examples

```
osm push -c mybucket f1.txt /tmp/f1.txt
```

### Options

```
  -c, --container string   Name of container
      --context string     Name of osmconfig context to use
  -h, --help               help for push
```
