## osm rc

Remove container

### Synopsis


Remove container

```
osm rc <name> [flags]
```

### Examples

```
osm rc mybucket
```

### Options

```
      --context string   Name of osmconfig context to use
  -f, --force            Force delete any files inside the container
  -h, --help             help for rc
```
