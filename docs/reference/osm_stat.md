## osm stat

Stat item from container

### Synopsis


Stat item from container

```
osm stat <id> [flags]
```

### Examples

```
osm stat -c mybucket f1.txt
```

### Options

```
  -c, --container string   Name of container
      --context string     Name of osmconfig context to use
  -h, --help               help for stat
```
