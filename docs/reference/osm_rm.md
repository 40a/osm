## osm rm

Remove item from container

### Synopsis


Remove item from container

```
osm rm <id> [flags]
```

### Examples

```
osm rm -c mybucket f1.txt
```

### Options

```
  -c, --container string   Name of container
      --context string     Name of osmconfig context to use
  -h, --help               help for rm
```
